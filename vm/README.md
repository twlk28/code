DaPU的设定
总线16, cpu字长16Bit(一条指令是 2 字节，一个地址也是 2 字节)

内存的设定
内存字长16Bit = 2bytes
内存寻址范围2字节 = 2**16 = 64K
内存容量 = 寻址范围*字长 = 64K * 2bytes = 128K bytes

内存区域划分的设计
头4个字节: 固定指令`jump @1024`
5-511: 存放字体文件
512-1023: 作为显存表示 32x16 像素大小的屏幕
1024-65535: 程序






字体文件:
一个字符4字节来存

字符编码(dacoding):
0      .
1      ,
2      空格
3      $
4      ￥
5-14    0-9
